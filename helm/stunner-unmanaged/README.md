# STUNner


Note that this chart is obsolete and will be removed in a future release. The stable, tested, supported and recommended way to install the STUNner ecosystem is by using the [STUNner chart](https://artifacthub.io/packages/helm/stunner/stunner). If you don't have any particular reason to use this chart, please use the mentioned chart.

The STUNner chart is available on [ArtifactHub](https://artifacthub.io/packages/helm/stunner/stunner), in its official [GitHub repository](https://github.com/l7mp/stunner-helm/tree/main/helm/stunner).
